## Getting Started

install node_modules using => npm install

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Dev and dependencies

```bash
 - prisma
 - @prisma/client
 - next-auth
 - railway
 - tailwindcss

 # Binding prisma with @next-auth/prisma-adapter
 $ npm install @next-auth/prisma-adapter
```
